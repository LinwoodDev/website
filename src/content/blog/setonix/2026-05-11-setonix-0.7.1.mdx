Setonix 0.7.1 is here with important launch fixes and a cleaner home page experience.
This release focuses on making the app open reliably across all supported platforms while improving the first screen users see.

Read the [big 0.7.0 release announcement](https://linwood.dev/setonix/0.7.0) for more details on the major features added in the previous release.

## Highlights

- Improved home page interface
- Launch fixes across platforms

## Improved home page interface \{#home-page\}

The home page user interface has been improved to make it easier to get started and navigate into your projects and add new colorful and descriptive icons to the home page. This should make it more welcoming and intuitive for new users.

## Launch fixes across platforms \{#launch-fixes\}

This release fixes an issue that prevented the app from launching correctly on iOS, macOS, Android, and web.
If you were unable to open Setonix on one of these platforms, please update to this version.

Read more on the [Setonix 0.7.1 release page](https://linwood.dev/setonix/0.7.1).

## Full changelog

* Improve user interface on home page
* Fix app not launching on iOS, macOS, Android and web
