This is a release candidate for the upcoming Butterfly 2.4.2 stable release. Please test this release candidate and report any issues you find.

## UI improvements

The UI of the PDF export dialog and the image export preview has been improved. Also, empty page names in the PDF select dialog now show "Untitled".

## Performance improvements

The performance of the eraser and path eraser tool has been improved. Also, baking on the pen tool is now skipped if a new element is started.

## Translations

Indonesian translation has been added. Also, style classes are now translated.

## Bug fixes

Several bugs have been fixed, including crashes when changing quality during PDF export, incorrect display of transformed images, and RTL text direction issues in the label and color toolbar. Also, the math button in the label toolbar now works correctly even if no text is selected.

## Full changelog
