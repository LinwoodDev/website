The fifth release candidate of Butterfly Black Hairstreak (2.4) is now available with many bug fixes.

## Bug fixes

On some devices the toolbar was overlapping with system elements, this has been fixed by adding a safe area to the toolbar.
Additionally the refresh button in the files view was duplicated, this has been fixed as well and the alignment of the create buttons in the pages navigator page has been fixed, aligning wrongly on the left.
Moreover, a freeze when importing files has been fixed and a crash when having large pdf to import has been fixed as well.
Furthermore, pages having problems since a long time where having the same name could lead to errors and duplicated pages in some cases, this has been fixed as well.


## Full changelog

* Add safearea on toolbar
* Fix duplicated refresh button in files view
* Fix alignment of create buttons in pages navigator page
* Fix freeze when importing
* Fix crash when having large pdf to import
* Fix error if pages have same name
* Fix problem in save system leading to duplicated pages in some cases
* Revert 16kb page size support effort
