We are now in the release candidate phase. This means that we will only fix bugs and not add new features. If you find a bug, please report it on [GitHub](https://linwood.dev/butterfly/report).

## Connection system

The connection system has been improved. You can now connect to a server with a self-signed certificate. This is useful if you want to use Butterfly on a local server. You can also add names to your connections to make it easier to identify them.
Additionally you can also disable certain directories types in the local storage.

## Full changelog

* Add files action to file navigator
* Add function to disable certain directory in local storage
* Add name to local storage
* Add extra properties for models to implement further import/export features
* Add option to connect to a self-signed server ([#531](https://github.com/LinwoodDev/Butterfly/issues/531))
* Improve web loading experience
* Fix changing current tool when adding in collaboration mode
* Fix remote connection requests
