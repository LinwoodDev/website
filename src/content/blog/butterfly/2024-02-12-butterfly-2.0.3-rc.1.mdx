A new release candiate is out now which adds initial ios support, adds an option to disable autosave and fixes many bugs. Thanks to [@fritzlb](https://github.com/fritzlb) for contributing to this release.
Click [here](/butterfly/2.0) to see the blog entry of the last 2.0 release and [here](/butterfly/2.0.1) to see the blog entry of the last 2.0.1 release.

## iOS support

We are happy to announce that we have added initial support for iOS. Currently, this is only available in the nightly builds and has known issues. Read more about it [here](https://github.com/LinwoodDev/Butterfly/issues/244).
Download it [here](https://docs.butterfly.linwood.dev/downloads/ios).

## Full changelog

* Add initial ios support (currently nightly only) ([#244](https://github.com/LinwoodDev/Butterfly/issues/244))
* Add option to disable autosave ([#591](https://github.com/LinwoodDev/Butterfly/issues/591))
* Convert aspect ratio menu to material 3
* Invert current inverted position to be normal
* Fix file sharing on android ([#585](https://github.com/LinwoodDev/Butterfly/issues/585))
* Fix laser movement is not locked
* Fix color of icon dropdown in property view
* Disable camera on android sdk version < 21
