Please note that this is still a beta release and may contain bugs. If you find a bug, please report it on [GitHub](https://linwood.dev/butterfly/report).

## Grid view

There is now a grid view in the home page. Click on the grid icon to change to this mode.
All file actions are stored in a popup menu.

## Elements

Click on label elements with the label painter to edit them.
Additionally you can now move elements in the z-axis with the hand tool.
