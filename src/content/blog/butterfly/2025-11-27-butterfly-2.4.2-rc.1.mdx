This is a release candidate for the upcoming Butterfly 2.4.2 stable release. Please test this release candidate and report any issues you find.

## Performance improvements

Performance of WebDAV operations and the bake queue has been improved. Additionally, baking now looks better, has better performance and unnecessary bakes without changes have been prevented.

## UI improvements

RTL layout issues with icons have been fixed, and the pack name is now correctly shown in the select asset dialog and the components view.

## Bug fixes

Several bugs have been fixed in this release, including smooth navigation, stamp handler toolbar issues.
Also, issues with saving pages when adding a new page and rotating the ruler in touch mode have been resolved.
