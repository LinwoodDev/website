The third release candidate of Butterfly Crimson Red (2.5) is now available.

Highlights:

- 📄 Improved PDF rendering
- ⌨️ Presentation shortcut
- ⚙️ Improved UI and Settings
- 📱 Template improvements
- 🐛 Bug fixes

Butterfly Crimson Red (2.5) is now close to the final stable release.
Please test this release candidate and report any remaining bugs you find.

## Improved PDF rendering

PDFs should now render more reliably.
There is also fixes with the PDF export dialog, such as overflow on mobile devices and thumbnails not loading.

## Presentation shortcut

The presentation toggle has been added to the keyboard shortcuts settings page and its default shortcut was changed to `Ctrl + Space` for quicker access.
Previously it conflicted with the spacebar input in text fields, which has now been fixed.

## Improved UI and Settings

On mobile, we added a warning to the data folder setting, and you'll now get a confirmation dialog when changing your data folder. Additionally, the Arabic font has been changed from Readex Pro to Noto Sans Arabic.

## Template improvements

The template selection UI has been improved for a smoother experience, adding correct padding on mobile and ensuring selection checkboxes are only visible in selection mode.

## Bug fixes

A fix where polygons would disappear after editing them has been implemented.

## Full changelog

* Improve pdf rendering
* Add toggle presentation to keyboard shortcuts settings page
* Change toggle presentation to ctrl + space
* Fix space not working in text input fields
* Fix pdf export dialog overflow on mobile devices
* Fix thumbnails not loading in pdf export dialog
* Fix polygon disappearing after editing
* Add warning to data folder setting on mobile
* Add confirmation dialog when changing data folder
* Replace Readex Pro with Noto Sans Arabic
* Improve template selection UI
  * Fix template selection padding on mobile
  * Hide selection checkboxes when not in selection mode
