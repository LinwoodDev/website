We are now in the release candidate phase. This means that we will only fix bugs and not add new features. If you find a bug, please report it on [GitHub](https://linwood.dev/butterfly/report).

## Input

You can now right click as alternative to long click to open the properties of the tools in the toolbar or edit the color in the color toolbar.

## Full changelog

* Add right click as alternative to long press
* Change default scale of label to 5
* Improve documentation of shortcuts ([#534](https://github.com/LinwoodDev/Butterfly/issues/534))
